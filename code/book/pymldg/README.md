* (+) 코드 예제 중심으로 내용을 이해할 수 있음.

* (-) 간결한 이론적 설명이 좋으나 자세함이 부족함.
    * 용어의 통일성 부족 & 장황한 문장 (간혹 있음)
    * 같은 내용의 반복이 빈번 (독자의 이해력을 높이기 위한 설명 방식일 수 있음)
    * 혼동되는 부분이 있어 다시 정리함 (본인의 이해력 부족일 수 있음)
